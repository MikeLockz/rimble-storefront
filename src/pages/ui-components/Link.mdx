# Link

The `Link` component provides a basic styled link.

**Import component**
```lang=jsx
import { Link } from 'rimble-ui';
```

## Example

```js is=react-live
<Link
href="#!"
target="_blank"
title="This link goes somewhere"

>

Click me!
</Link>
```
