# Avatar documentation

A basic avatar component

_Want to understand best practices for using`Avatar`?_ [Read design guidelines](/ui-components/avatar/design)

Import component

```lang=jsx
import { Avatar } from 'rimble-ui';
```

## Examples

Different ways to use the `Avatar` component

### Standard avatar

```js is=react-live
<Avatar />
```

### Standard with image

```js is=react-live
<Avatar src="[https://airswap-token-images.s3.amazonaws.com/DAI.png](https://airswap-token-images.s3.amazonaws.com/DAI.png)" />
```

### Custom size

```js is=react-live
<Avatar
  size="100px"
  src="[https://airswap-token-images.s3.amazonaws.com/DAI.png](https://airswap-token-images.s3.amazonaws.com/DAI.png)"
/>
```
