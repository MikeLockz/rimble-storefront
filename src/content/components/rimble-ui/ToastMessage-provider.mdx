# ToastMessage.provider

**mount `<ToastMessage.Provider />`**

```lang=jsx
<ToastMessage.Provider ref={node => (window.toastProvider = node)} />
```

## Previews

Preview each `ToastMessage` and see how to call it.

**addMessage.processing**

Calls a processing `ToastMessage`.

```js is=react-live
<Button
  mb={3}
  onClick={e =>
    window.toastProvider.addMessage("Processing payment...", {
      secondaryMessage: "Check progress on Etherscan",
      actionHref:
        "[https://etherscan.io/tx/0xcbc921418c360b03b96585ae16f906cbd48c8d6c2cc7b82c6db430390a9fcfed](https://etherscan.io/tx/0xcbc921418c360b03b96585ae16f906cbd48c8d6c2cc7b82c6db430390a9fcfed)",
      actionText: "Check",
      variant: "processing"
    })
  }
>
  Preview
</Button>
```

**addMessage.success**

Calls a success `ToastMessage`.

```js is=react-live
<Button
  mb={3}
  onClick={e =>
    window.toastProvider.addMessage("ETH sent", {
      secondaryMessage: "You have 1.03 Ether remaining",
      variant: "success"
    })
  }
>
  Preview
</Button>
```

**addMessage.failure**

Calls a failure `ToastMessage`.

```js is=react-live
<Button
  mb={3}
  onClick={e =>
    window.toastProvider.addMessage("Payment failed", {
      secondaryMessage: "You don't have enough Ether (ETH)",
      actionHref: "[https://www.coinbase.com/](https://www.coinbase.com/)",
      actionText: "Buy",
      variant: "failure"
    })
  }
>
  Preview
</Button>
```

**addMessage.default**

**addMessage.default with icon**

**addMessage.default dark colour with icon**

Calls a default dark `ToastMessage` with an icon.

  Preview

**removeMessage**

Removes a `ToastMessage`.

```js is=react-live
<Button
mb={3}
onClick={e =>
window.toastProvider.addMessage('Welcome to my dApp', {
icon: 'Mood',
colorTheme: 'dark',
})
}

>

Call toast
</Button>

<Button onClick={() => window.toastProvider.removeMessage()}>
```
