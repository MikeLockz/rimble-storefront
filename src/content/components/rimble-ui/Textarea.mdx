The `Textarea` component provides a styled version of the HTML `<textarea>`.

**Import component**

```lang=jsx
import { Textarea } from 'rimble-ui';
```

## Example

```js is=react-live
<Textarea placeholder="Start typing..." rows={4} />
```

** Styled System Props **

`Textarea` uses `COMMON`, `LAYOUT`, `POSITION`, `FLEXBOX`, `BORDERS` groups of [Styled System props](/components/rimble-ui/system-props).
