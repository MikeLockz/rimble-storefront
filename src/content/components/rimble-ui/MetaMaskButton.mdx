Use the `MetaMaskButton` component to let users know they can connect to your dapp using MetaMask. `MetaMaskButton` maps to a `<button type='button'>` element and inherits props from [`Button`](/components/rimble-ui/Button).

*Want to understand best practices for using a button component?* Read [Button design guidelines](/components/rimble-ui/Button#design-guidelines)

**Import component**

```
import { MetaMaskButton } from 'rimble-ui';
```

## Examples

Different ways to use the `MetaMaskButton` component in your product.

**Default**

```js is=react-live
<MetaMaskButton>Connect with MetaMask</MetaMaskButton>
```

**Outline**

```js is=react-live
<MetaMaskButton.Outline>Connect with MetaMask</MetaMaskButton.Outline>
```

**Different sizes**

```js is=react-live
<MetaMaskButton size="small">Connect with MetaMask</MetaMaskButton>
```

```js is=react-live
<MetaMaskButton.Outline size="small">
  Connect with MetaMask
</MetaMaskButton.Outline>
```

```js is=react-live
<MetaMaskButton.Outline size="medium">
  Connect with MetaMask
</MetaMaskButton.Outline>
```

```js is=react-live
<MetaMaskButton size="large">Connect with MetaMask</MetaMaskButton>
```

```js is=react-live
<MetaMaskButton.Outline size="large">
  Connect with MetaMask
</MetaMaskButton.Outline>
```

**Full-width button**

```js is=react-live
<MetaMaskButton width={1}>Connect with MetaMask</MetaMaskButton>
```

```js is=react-live
<MetaMaskButton.Outline width={1}>Connect with MetaMask</MetaMaskButton.Outline>
```

**Disabled**

```js is=react-live
<MetaMaskButton disabled>Connect with MetaMask</MetaMaskButton>
```

```js is=react-live
<MetaMaskButton.Outline disabled>Connect with MetaMask</MetaMaskButton.Outline>
```

## Props

<PropsTable propMetaData={props.propMetaData} />

`MetaMaskButton` inherits all props from the [`Button`](/components/rimble-ui/Button#props) component.

** Styled System Props **

`MetaMaskButton` uses `COMMON`, `LAYOUT`, `POSITION`, `FLEXBOX`, `BORDERS` groups of [Styed System props](/components/rimble-ui/system-props).
