The `Flex` component renders a responsive flexbox layout component that controls the layout of the components nested inside. Use it when you want to create mobile responsive layouts or want to arrange content in columns or rows. This component extends [`Box`](/components/rimble-ui/Box).

**Import component**

```lang=jsx
import { Flex, Box } from 'rimble-ui';
```

## Example

```js is=react-live
<Flex>
  <Box p={3} width={1 / 2} color="salmon" bg="black">
    Flex
  </Box>
  <Box p={3} width={1 / 2} color="white" bg="salmon">
    Box
  </Box>
</Flex>
```
