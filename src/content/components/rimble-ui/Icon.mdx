The Icon component renders Material Design Icons. Built with [rmdi](https://github.com/jxnblk/rmdi).

**Import component**

```lang=jsx
import { Icon } from 'rimble-ui';
```

## Examples

**Standard**

```js is=react-live
<Icon name="MoneyOff" />
```

**Custom colour**

```js is=react-live
<Icon color="primary" name="MoneyOff" />
```

**Custom size**

```js is=react-live
<Icon color="tomato" name="MoneyOff" size="80" />
```
