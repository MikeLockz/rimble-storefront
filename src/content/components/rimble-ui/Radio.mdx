Use the `Radio` component when users can select only one option from a short list and you want all the options to be visible. `Radio` maps to a radio input and a label.

**Import component**

```lang=jsx
import { Radio } from 'rimble-ui';
```

## Example

```js is=react-live
<Field label="Choose transaction speed">
  <Radio label="Quick" my={2} />
  <Radio defaultChecked label="Standard" my={2} />
  <Radio disabled label="Slower" my={2} />
  <Radio checked disabled label="Custom" my={2} />
</Field>
```

## Design guidelines

Some best practices for using `Radio` in your product.

**Always use with a label or heading**

Your users will need instruction on what to choose. Don't just provide them with their choices.

<DoDont>
  {{
    positive: (
      <Box>
        <Field label="Would you like alerts when the price changes?">
          <Radio label="Yes" my={2} />
          <Radio label="No" my={2} />
        </Field>
      </Box>
    ),
    negative: (
      <Box>
        <Radio label="Give me price changes" my={2} />
        <Radio label="Don't give me price changes" my={2} />
      </Box>
    )
  }}
</DoDont>

**Only use for binary decisions**

Radio buttons are for times when you users can choose only one option.

<DoDont>
  {{
    positive: (
      <Box>
        <Field label="Would you like alerts when the price changes?">
          <Radio label="Yes" my={2} />
          <Radio label="No" my={2} />
        </Field>
      </Box>
    ),
    negative: (
      <Box>
        <Field label="Choose any currencies you'd like to track">
          <Radio label="Ether (ETH)" my={2} />
          <Radio label="Bitcoin (BTC)" my={2} />
          <Radio label="Ethereum Classic (ETC)" my={2} />
          <Radio label="Litecoin (LTC)" my={2} />
        </Field>
      </Box>
    )
  }}
</DoDont>

**Only use for small groups of options**

If the user can choose from lots of options, like currencies or states, use a [Select](/components/rimble-ui/Select) component instead.

<DoDont>
  {{
    positive: (
      <Box>
        <Field label="Choose your default currency">
          <Select
            options={[
              { value: "eth", label: "Ethereum" },
              { value: "btc", label: "Bitcoin" },
              { value: "gno", label: "Gnosis" },
              { value: "gnt", label: "Golem" },
              { value: "rep", label: "Augur" }
            ]}
          />
        </Field>
      </Box>
    ),
    negative: (
      <Box>
        <Field required={true} label="Choose your default currency">
          <Radio label="ETH - Ether" my={2} />
          <Radio label="BTC - Bitcoin" my={2} />
          <Radio label="USD - Dollars" my={2} />
          <Radio label="GBP - Pounds" my={2} />
          <Radio label="XRP - Ripple" my={2} />
          <Radio label="GNO - Gnosis" my={2} />
          <Radio label="GNT - Golem" my={2} />
          <Radio label="REP - Augur" my={2} />
          <Radio label="YEN" my={2} />
          <Radio label="INR - Rupee" my={2} />
        </Field>
      </Box>
    )
  }}
</DoDont>
