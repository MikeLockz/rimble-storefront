# Slider

The `Slider` component provides a styled slider control for the HTML `<input type="range">`.

**Import component**

```lang=jsx
import { Slider } from 'rimble-ui';
```

## Example

```js is=react-live
<Slider min={"0"} max={"100"} step={"10"} />
```
