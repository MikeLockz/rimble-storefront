Use the `QR` component to renders a QR code for a provided string. Can be used as a shortcut to input a public Ethereum address or link to a specific website.

**Import component**

```
import { QR } from 'rimble-ui';
```

## Example

```js is=react-live
<QR value="0xAc03BB73b6a9e108530AFf4Df5077c2B3D481e5A" />
```

**Setting custom size by pixels**

```js is=react-live
<QR value="0xAc03BB73b6a9e108530AFf4Df5077c2B3D481e5A" size={40} />
```

**Setting custom size by percent**

```js is=react-live
<QR value="0xAc03BB73b6a9e108530AFf4Df5077c2B3D481e5A" size={"100%"} />
```

## Props

<PropsTable propMetaData={props.propMetaData} />

The `QR` component uses [react.qrcode](https://www.npmjs.com/package/qrcode.react) to generate the QR code from the _value_ prop.
