A wrapping component that applies global styles. Use the Base Styles component in conjunction with the [ThemeProvider](/components/rimble-ui/ThemeProvider) component to apply css resets, default typography, and other globally-defined styles.

**Import component**

```jsx
import { BaseStyles, ThemeProvider, theme } from "rimble-ui";
```

## Examples

**Apply default global styles**

```js is=react-live
<ThemeProvider>
  <BaseStyles>
    <Button>Default Rimble theme</Button>
  </BaseStyles>
</ThemeProvider>
```

**Override global styles**

```js is=react-live
<ThemeProvider
  theme={Object.assign({}, theme, {
    colors: {
      ...theme.colors,
      white: "black",
      background: "blue",
      primary: "crimson"
    }
  })}
>
  <BaseStyles>
    <Button>Override Rimble theme</Button>
  </BaseStyles>
</ThemeProvider>
```

## Design guidelines

Some best practice for using `BaseStyles` in your product.

**Use with ThemeProvider**

ThemeProvider passes in variables to set sizes for typography and spacing that gets applied globally via the BaseStyles.
