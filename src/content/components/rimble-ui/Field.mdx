# Field

The `Field` component provides a convenient way to add a label to an [`Input`](/components/rimble-ui/Input) component.

**Import component**

```lang=jsx
import { Field } from 'rimble-ui';
```

## Example

```js is=react-live
<Field label="Public address">
  <Input type="text" required="true" />
</Field>
```
