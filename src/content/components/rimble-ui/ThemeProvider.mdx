A wrapping component that overrides default component styles. ThemeProvider allows all child components to have access to the theme object via the Context API. [Learn more about ThemeProvider from Styled-Components](https://www.styled-components.com/docs/advanced#theming).

**Import component**

```jsx
import { BaseStyles, theme } from "rimble-ui";
import { ThemeProvider } from "styled-components";
```

## Examples

**Apply default global styles**

```js is=react-live
<ThemeProvider>
  <BaseStyles>
    <Button>Default Rimble theme</Button>
  </BaseStyles>
</ThemeProvider>
```

**Override global styles**

```js is=react-live
<ThemeProvider
  theme={Object.assign({}, theme, {
    colors: {
      ...theme.colors,
      white: "black",
      background: "blue",
      primary: "crimson"
    }
  })}
>
  <BaseStyles>
    <Button>Override Rimble theme</Button>
  </BaseStyles>
</ThemeProvider>
```

## Design guidelines

Some best practice for using `BaseStyles` in your product.

**Use with ThemeProvider**

ThemeProvider passes in variables to set sizes for typography and spacing that gets applied globally via the BaseStyles.
