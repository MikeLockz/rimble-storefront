ThemeProvider allows for overriding the default component styles holistically from a single theme definition.

# Theme Provider

Rimble components use [styled-system](https://github.com/jxnblk/styled-system) to support theme-based variables.

**Import component**

```lang=jsx
import { ThemeProvider } from 'styled-components'
```

## Example

**Default theme**

```js is=react-live
<ThemeProvider>
  <Button>Button</Button>
</ThemeProvider>
```

**Custom theme**

```js is=react-live
<ThemeProvider theme={myTheme}>
  <Button>Button</Button>
</ThemeProvider>
```
