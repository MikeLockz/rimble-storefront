# Select

Use the Select component when you want your users to select an option from a list. Select provides a styled version of the HTML `<select>`. This component is specifically useful for long lists you wouldn't want to display using [`Radio`](/ui-components/Radio) buttons like currencies or states.

**Import component**

```lang=jsx
import { Select } from 'rimble-ui';
```

## Example

```js is=react-live
<Field label="Choose your currency">
  <Select
    items={[
      "ETH - Ether",
      "BTC - Bitcoin",
      "GNO - Gnosis",
      "GNT - Golem",
      "REP - Augur"
    ]}
    required="false"
  />
</Field>
```

## Design guidelines

Some best practices for using `Select` in your product.

**It's better to show your options**

Only use the `Select` component when you have a lot of options that would clutter your interface. If you've only got a few options to choose from it might be better to show them to the user with a [`Radio`](/Develop/Radio) component.
