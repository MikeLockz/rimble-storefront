## Installation
Install `rimble-ui` and `styled-components` in your project with npm:

```
npm install --save rimble-ui styled-components
```

or with yarn:

```
yarn add rimble-ui styled-components
```

## Usage

Play with an example on <Link href="https://codesandbox.io/embed/little-meadow-m3sv3" target="_blank">CodeSandbox</Link>

The code below will render a button with default Rimble styles.

```jsx
import React from "react";
import ReactDOM from "react-dom";
import { Button } from 'rimble-ui';

function App() {
  return (
    <div>
      <Button> click me! </Button>
    </div>
  );
}

const rootElement = document.getElementById("root");
ReactDOM.render(<App />, rootElement);
```
