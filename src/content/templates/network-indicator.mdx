The network indicator will passively tell the user about the network they're on and whether that's correct. This should be used in conjunction with a [Connection banner](/components/web3-components/ConnectionBanner) to provide more context and information.

When a user goes to transact and must be on a certain network to transact successfully, you should also use a [Wrong network modal](/templates/wrong-network) to help explain what they need to do.
